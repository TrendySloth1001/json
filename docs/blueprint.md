# **App Name**: JSON Brilliance

## Core Features:

- Raw JSON Input: Text area for users to paste raw JSON.
- Prettify JSON: Formats the JSON and displays it in a structured, readable format.
- Syntax Highlighting: Highlights JSON elements with distinct colors for better readability.
- Theme Switcher: Allows users to toggle between 'Anime' (energetic pastel neon) and 'Light' (clean minimalist) themes.
- Error Detection: Identifies and highlights errors in the raw JSON input, providing a descriptive error message.
- Minify JSON: Compresses JSON to remove unnecessary characters for efficient storage or transfer.
- AI powered schema suggestor: Suggest a schema definition as a tool that checks for errors in new edits and provides contextual information based on best practices and data consistency rules.

## Style Guidelines:

- Primary color for Anime Theme: Vibrant pink (#FF69B4) for a playful and energetic feel.
- Background color for Anime Theme: Very light pink (#F8BBD0) for an energetic feel.
- Accent color for Anime Theme: Electric purple (#BE29EC) for interactive elements.
- Primary color for Light Theme: Light gray (#E0E0E0) for a clean, minimalist look.
- Background color for Light Theme: White (#FFFFFF) for maximum readability and simplicity.
- Accent color for Light Theme: Dark blue (#1976D2) for buttons and active states.
- Body and headline font: 'Inter', sans-serif, for readability.
- Simple, intuitive icons for actions like 'Prettify', 'Minify', and theme switching.
- Split-screen layout with resizable panels for raw JSON input and formatted output.
- Smooth transitions when switching between themes and formatting JSON.
